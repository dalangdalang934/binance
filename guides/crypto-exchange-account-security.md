# Crypto Exchange Account Security

Exchange account security should be completed before depositing funds. Most beginner losses are caused by phishing, weak passwords, fake apps, SIM swap risk, or sending assets to the wrong network.

## Minimum Security Setup

- Use a unique password.
- Enable two-factor authentication with an authenticator app or hardware key.
- Set an anti-phishing code for exchange emails if supported.
- Add withdrawal address allowlists when available.
- Keep recovery codes offline.
- Do not share screenshots that reveal account identifiers.

## Deposit and Withdrawal Safety

- Send a small test transfer first when using a new address.
- Match the asset and network exactly.
- Watch for copied-address malware.
- Keep records of transaction IDs.
- Understand withdrawal fees before transferring.
