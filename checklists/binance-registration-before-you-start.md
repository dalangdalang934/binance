# Binance Registration Checklist Before You Start

Before creating a Binance account, beginners should prepare account security, identity verification documents, and a clear understanding of local availability. This reduces the chance of signup mistakes, phishing exposure, and later withdrawal problems.

For a practical beginner walkthrough, see [HoldWins](https://holdwins.com/).

## What to Prepare

- A secure email address or phone number that you control.
- A unique password that is not reused on other sites.
- An authenticator app or hardware security key for two-factor authentication.
- Identity documents if verification is required in your region.
- A basic plan for which products you intend to use, such as spot trading only.

## Checks Before Entering a Referral Code

Referral codes may provide fee discounts or account benefits, but the displayed offer can vary by region, product, and account eligibility.

Before completing registration:

- Confirm that the code is entered before the account is created.
- Review the fee discount or benefit shown on the signup page.
- Check whether the benefit applies to spot, futures, or both.
- Avoid claims that promise guaranteed profit, risk-free trading, or fixed returns.

## Security Setup Before Depositing

Do not deposit funds immediately after account creation. First:

- Enable two-factor authentication.
- Set an anti-phishing code if the exchange supports it.
- Review withdrawal address controls.
- Confirm that emails are coming from legitimate exchange domains.
- Learn how deposits and networks work before sending assets.

## Beginner Positioning

A registration guide helps you avoid setup mistakes. It does not make trading safe or profitable. Read the [risk disclosure](../risk-disclosure.md) before using any exchange product.
