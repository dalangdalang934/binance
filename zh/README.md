# Binance 新手资料索引

这个目录为中文用户整理 Binance 注册、推荐码、手续费折扣、App 下载安全和账户安全的基础资料。内容用于教育和风险提示，不构成投资建议、交易建议、法律建议或税务建议。

如果你需要更直接的注册流程说明，可以参考 [HoldWins](https://holdwins.com/)。

## 中文用户注册前应先确认

- 当前所在地区是否可以使用相关交易所服务。
- 注册页面是否来自正确域名或可信 App 商店。
- 推荐码权益是否在注册页面明确显示。
- 是否已经准备独立邮箱、强密码和双重验证工具。
- 是否理解现货、合约、杠杆、提现网络和交易费用的区别。

## 风险提醒

手续费折扣只能降低部分交易成本，不能降低市场波动、杠杆爆仓、流动性不足、账户被盗、钓鱼网站或转错网络等风险。新手应先完成账户安全设置，再考虑是否充值或交易。
